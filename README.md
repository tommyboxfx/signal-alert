# PocketOption Signal Alert

Instant sound notification for trading signals.

## Download

Go to [Releases](https://github.com/tommyboxfx/signal-alert/releases) and download the latest `.exe`.

## How to use

1. Download `Signal_alert_v1.4.exe`
2. Run it
3. Choose your alert sound
4. Click **Connect**
5. Done! You'll hear the alert when a signal arrives.

## Features

- 8 alert sounds: Siren, Ship Horn, Bell, Alarm, Triple Beep, Air Raid, Foghorn, Bugle
- Volume slider
- Windows popup notification
- Auto-reconnect
- Payout filter (skips pairs below 75%)
- Zero configuration needed
